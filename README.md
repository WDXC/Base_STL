# Base_STL
一个使用C++11去完成STL的项目
主要内容包括:
1. 空间配置器
2. 迭代器
3. 容器
	* vector
	* list
	* deque
	* queue
	* stack
	* RB-Tree
	* set
	* map
	* multimap
	* multiset
	* Hash-Table
	* Hash-Set
	* Hash-Map

4. 算法
	* 数值算法
	* 基本算法
	* 其它算法

5. 仿函数
6. 配接器

# 编译方式
Debug 版本  
```
./build.sh debug
```

Release 版本
```
./build.sh release
```


# 项目计划
本项目是基于候捷老师的《STL源码分析》中的SGI STL，使用C++11标准对其更新重构。  
项目不一定会按照书中的要求进行排序，预计项目用时4个月完成。
